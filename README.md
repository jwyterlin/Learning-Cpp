# Learning C++
In this project I will make some little programs to learn and improve my knowledges about C++.

## Setup

```
$ git clone https://github.com/jwyterlin/Learning-Cpp.git
$ cd Learning-Cpp
```

## To run it locally

```
In Mac OS X:
$ g++ <filename>.cpp -o <filename>
$ ./<filename>
```
